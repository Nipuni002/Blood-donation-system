# Blood Donor Client

Lightweight React frontend (Vite) for the Blood Donor app.

Requirements
- Node.js >=16

Install and run (PowerShell):

```powershell
cd client
npm install
npm run dev
```

- Auth API: http://localhost:4000/auth
- Donor API: http://localhost:3000

Notes
- Register as `role` `user` or `admin`.
- Admins see the donor list and can delete donors.
